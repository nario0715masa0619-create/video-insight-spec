\# video-insight-spec：アホ社長向け説明



\## 今、あなたの会社が困ってること



\### 例：YouTube Analytics で見てること

「えっと、Python 講座が 38,000 回再生か。まあ、人気だな」



\### でも本当の問題

\- ❌ なぜ売れてるのか？ → 分からない

\- ❌ Web 制作講座の視聴数が少ないのはなぜ？ → 感覚で「つまらないんかな」

\- ❌ 競合の講座が成功してるってニュース見たけど、何が違うの？ → 分からない

\- ❌ AI 講座やるべき？→ トレンドだから？それとも？→ 分からない



→ つまり、根拠なく経営判断してる = 年間1億円の開発費が無駄になる可能性



\---



\## video-insight-spec が教えてくれること（シンプル版）



\### 3 つの質問に答える仕組み



\*\*質問 1：「この講座、何が強いの？」\*\*

\- YouTube Analytics ：「38,000 再生です」

\- video-insight-spec ：「初心者向けだから最後まで見る人が多い（完了率 68%）だから売れる」



\*\*質問 2：「なんで Web 制作は売れないの？」\*\*

\- YouTube Analytics ：「12,000 再生です」

\- video-insight-spec ：「基礎知識だけで実装がない。途中で見なくなる（完了率 32%）だから売れない」



\*\*質問 3：「競合の講座が成功してるのはなぜ？」\*\*

\- YouTube Analytics ：「知りません」

\- video-insight-spec ：「『基礎 → 実装 → 実務案件』の 3 段階だから実務で使える」



\---



\## つまり、video-insight-spec は「採点表」



\*\*YouTube Analytics（今のあなた）：\*\*

\- 国語：80点 ← 点数だけ

\- 数学：70点 ← 何を直せばいいか分からない



\*\*video-insight-spec（新しい方法）：\*\*

\- 国語：80点

&#x20; - 読解力：90点 ← ここが強い

&#x20; - 作文力：50点 ← ここが弱い

&#x20; - 改善案：「毎日作文練習」

\- 数学：70点

&#x20; - 計算：85点 ← ここが強い

&#x20; - 文章題：45点 ← ここが弱い

&#x20; - 改善案：「文章題を 100 問解く」



→ 「強み・弱み・改善案」が全部分かる



\---



\## 実例で説明



\*\*◆ Python 講座（38,000 再生）\*\*

\- ❶ 「初心者向け」と「実装」がセット

\- ❷ 「完了率 68%」= 途中で飽きない

\- ❸ 「エンゲージメント 1.72%」= 実務で使える

\- 結論：「金のなる木」。このカテゴリに投資を増やすべき



\*\*◆ Web 制作講座（12,000 再生）\*\*

\- ❌ 「基礎知識だけ」で実装がない

\- ❌ 「完了率 32%」= 途中で飽きる

\- ❌ 「エンゲージメント 0.89%」= 役に立たない

\- 結論：「金の無駄」。リメイクするか廃止するか



\*\*◆ 競合 A 社の Python 講座（45,000 再生）\*\*

\- ✅ 「基礎 → 実装 → 実務案件」の 3 段階

\- ✅ 「完了率 72%」= つまずきが少ない

\- ✅ 「成長率 +28%」= 市場で話題になってる

\- 結論：「本物のカリキュラム」。うちもこの形にリメイク

\## 社長が知りたい 4 つのこと



\*\*1. うちの強みは？\*\*

\- ❌ 今：「なんとなく Python が人気」

\- ✅ video-insight-spec：「初心者向けの『完了率 68%』が最強」

\- → 経営判断が明確になる



\*\*2. ライバルに負けてるのはなぜ？\*\*

\- ❌ 今：「競合の講座が良いらしい」

\- ✅ video-insight-spec：「『基礎 20% + 実装 60% + 実務 20%』の比率だから」

\- → 何を直せばいいか明確になる



\*\*3. 投資すべきカテゴリはどれ？\*\*

\- ❌ 今：「AI が流行ってるから AI やろう」

\- ✅ video-insight-spec：「Python は +22% で AI は +12%。2 倍成長」

\- → 根拠のある経営判断ができる



\*\*4. 改善したら、本当に売上増える？\*\*

\- ❌ 今：「改善したら売上増えるといいな」

\- ✅ video-insight-spec：「リメイクしたら売上 +95% の見込み」

\- → ROI が計算できる
