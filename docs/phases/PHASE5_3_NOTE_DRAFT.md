# 動画の視聴データから、経営判断に効く「実行可能な知恵」を毎月1枚のレポートに。

**作成日**: 2026-03-27  
**ステータス**: Draft（見出し + ラフ文）

---

## イントロ / Opening

動画コンテンツで事業をしている人なら、こんな悩みはないでしょうか。

「毎月の視聴ログは取れているけど、それがビジネスにどう活かせばいいのか分からない」

Excel で講座ごとの視聴数を集計したり、テーマごとの成長率を計算したり…正直、これって時間がかかる割に、「次に何をすべきか」が曖昧なままになりがちです。

このプロジェクトは、そんな課題をデータの構造設計 + レポート自動生成で解決したものです。

---

## 背景：「なぜこんなことを作ったのか」

EdTech / オンライン教育の企業で働いていた時、毎月の売上分析を営業チームにレポートしていました。

でも、データをいくら提示しても、営業側が「では、どの講座に注力すべきか」という判断に落とし込むまでに、もう一段階の説明が必要だった。

データはあるのに、「経営判断」に直結しないギャップがそこにありました。

その時思ったのが「もし、JSON の構造から レポート生成まで、すべてが一貫して設計されていたら、データと判断のギャップはなくなるはずだ」ということです。

---

## なぜ「動画 → JSON → 3レイヤー」という設計にしたのか

動画コンテンツの情報構造を、見えやすく整理する必要がありました。

**【Layer 1: video_meta】** 講座 ID、タイトル、難度、分類テーマ — 「講座とは何か」の基本定義

**【Layer 2: knowledge_core】** 各講座内の重要ポイント（Representative Pin）— 「その講座から何を学べるか」の本質

**【Layer 3: views】** Portfolio View（全講座）、Growth View（成長トレンド）、Theme View（テーマ別分析）— 「視聴者がどう反応しているか」

この 3 層を分離することで「何の講座か」「どんな価値か」「どう売れているか」が独立して見える。

そしてそれらを組み合わせると、初めて「経営判断に必要な情報」が浮かぶようになるんです。

---

## どんなレポートが出るのか（Executive Summary + Full Report の実例）

設計したレポートは 2 段階です。

**第 1 段階：Executive Summary（1 枚）**

社長が 3 分で読めるサイズ。総ビュー数、平均エンゲージメント、成長テーマ数。そして「Top 3 講座」と「重要テーマ ×3」を理由付きで表示。

例えば、こんな具合です：

- 総ビュー数：246,629
- 平均エンゲージメント率：1.71%
- 成長テーマ数：3 つ

**Top 3 講座：**
1. Lecture 01（118,000 views, 1.61% 高エンゲージ）— 「基礎ニーズが継続的に強い」
2. Lecture 02（47,739 views, 1.68%）— 「着実な成長を見せている」
3. Lecture 03（33,800 views, 1.89%）— 「視聴者の関心が高い」

**重要テーマ ×3：**
1. マーケティング — 全体を牽引する最優先テーマ
2. Web マーケティング — ボリュームゾーン。継続強化推奨
3. 自社分析 — 成長ポテンシャルのあるテーマ

これだけで、経営層は「今月は何をすべきか」が見えてきます。

**第 2 段階：Full Report（詳細版）**

Portfolio View で全講座の詳細メトリクス、Growth View で 1 週単位の推移、Theme View でテーマ別の Representative Insight を掲載。

施策を立案するための根拠データがすべて揃っています。

## どうビジネスになるのか（¥10万/月のサブスクリプション構想）

「データの構造」と「レポート」ができたら、それをサービス化する道が見えてきました。

**講座数 5～20 本クラスの講座制作企業・EdTech スタートアップ向けに、月額 ¥100,000 で以下を提供します：**

- 初期導入費：¥300,000（データセットアップ、初回レポート生成）
- 毎月末：Executive Summary + Full Report を自動配信
- 必要に応じて：レポート内容の簡易オンライン説明（15～30分・任意）

**期待される効果：**

月 1～2 新テーマを発見でき、月 2～3 講座の改善施策が立案できるようになります。

仮に月あたりの売上インパクトが数十万〜百万円規模になれば、月額投資 ¥100,000 は十分に回収できるという計算です。

---

## 実装の工夫：HTML / Text デュアル出力

レポートは 2 形式で生成されます。

**HTML 版**：ブラウザで見やすく、グラフや表を含むデザイン性重視。

**Text 版**：メール添付や CMS 連携を想定した、シンプルな markdown 形式。

同じ JSON データから両形式を自動生成することで、運用コストを最小化しました。

## まとめ

動画事業者にとって「データはあるけど、経営判断に落とし込めない」という課題は、本質的には「情報設計」の問題です。

3 レイヤーの JSON 構造と、それに基づいた Executive Summary + Full Report により、その課題は大きく軽減できます。

月額 ¥100,000 というプライシングは、企業側が月 1～2 テーマの発見から得られる施策立案のコストを考えれば、十分に合理的。

今後も、講座数 5～20 本クラスの EdTech スタートアップや講座制作企業が、このレポートを活用してデータドリブンな意思決定ができるようになることを期待しています。
