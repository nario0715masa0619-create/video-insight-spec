## 1. ヒーロー / Hero

「動画の**視聴データ**から、経営に効く**実行可能な知恵**を毎月1枚のレポートに。」

**対象**: 講座制作企業・EdTech スタートアップ  
**投資**: 初期導入費 30万円～ / 月額 10万円～  
**効果**: 月 1～2 新テーマ発見、月 2～3 講座改善、施策実行率 60%+

---

## 2. ペイン → ソリューション

### ペイン（顧客課題）

- 視聴データはあるが、**アクション不明**（「次に何をすべき？」が分からない）
- 集計に **1～2週間かかる**（Excel、手作業）
- **テーマ別の成長が見えない**（個別講座ごとの分析は手作業）
- 機会ロスを放置している（成長テーマの深掘りができていない）

### ソリューション

- **Executive Summary（1枚）**: 社長が 3 分で読める月次レポート
- **Full Report（詳細）**: Portfolio View / Growth View / Theme View で施策立案の根拠を提供
- **必要に応じた簡易オンライン説明（任意）**: レポート内容のポイント解説（15～30分程度）

---

## 3. JSON・3レイヤー構造（テキスト図解）

\\\
【入力】動画・視聴ログ データ
         ↓
【Layer 1: video_meta】講座ID・タイトル・難度・テーマ
         ↓
【Layer 2: knowledge_core】ポイント・Representative Pin（代表的な学び）
         ↓
【Layer 3: views】Portfolio View / Growth View / Theme View
         ↓
【出力】Executive Summary + Full Report（1枚 + 詳細）
\\\

## 5. 料金・導入プロセス

### 料金体系

| 項目 | 金額 |
|------|------|
| 初期導入費 | ¥300,000～ |
| 月額（基本プラン） | ¥100,000～ |
| 対象講座数 | 5～20 講座（上限 ~50） |

※ 50 講座超は個別見積もり

### 導入ステップ

**ステップ 1: オンライン ヒアリング**
- 貴社の講座数・テーマ構成・課題を確認
- 分析対象の講座・期間を決定

**ステップ 2: データ連携・初回分析**
- API / CSV 形式でデータを送信
- 初期データセットアップ（1～2 週間）

**ステップ 3: 初回レポート納品**
- Executive Summary + Full Report を配信
- 初回レビュー（任意、簡易説明）

**ステップ 4: 月次レポート自動配信開始**
- 毎月末：レポート自動配信（メール + PDF/HTML）
- 必要に応じて：レポート内容の**簡易オンライン説明（15～30分程度・任意）**

---

## 6. セールスポイント（差別化要素）

- **3レイヤー精密分析**: video_meta → knowledge_core → views で、表層的でない根拠を提供
- **1枚で判断できる Executive Summary**: 社長が 3 分で読めて、施策判断を即座に実行可能
- **自動配信 + 簡易サポート**: 毎月の煩雑な集計不要。必要に応じて軽いオンライン対応
- **クイックスタート**: 最短 4～5 週間で初回レポート配信
