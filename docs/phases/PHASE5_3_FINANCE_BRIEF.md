## 1. 事業概要

### 事業名
**Video Insight Analytics**  
動画コンテンツの視聴データを構造化し、経営判断に直結したレポートを自動生成・配信するサブスクリプション型 SaaS サービス。

### 対象顧客
講座数 5～20 本クラスの講座制作企業・EdTech スタートアップ。  
年間売上 1～10 億円規模で、データドリブン改善に取り組みたいが、運用体制が十分でない企業層。

### 提供価値
- **データ構造化**: 3 レイヤー JSON（video_meta / knowledge_core / views）で複雑な動画ビジネスを可視化
- **自動レポート生成**: Executive Summary（1 枚）+ Full Report（詳細）で、経営判断と施策立案の時間を 80% 削減
- **継続的な改善提案**: 月 1～2 新テーマ発見、月 2～3 講座改善施策の立案が可能に

### 収益モデル
- **基本プラン**: 月額 ¥100,000 / 社（最小契約期間 3 ヶ月）
- **初期導入費**: ¥300,000 ～ ¥500,000（講座数・カスタマイズ度による）
- **将来オプション**: プレミアムプラン（月 ¥200k、週次報告）、エンタープライズ（要相談）

## 2. 市場機会

### 市場規模の推定

**グローバル EdTech 市場**

グローバル EdTech 市場は、2023 年時点でおよそ 2,000～2,500 億ドル規模と推計されており、  
2033 年には 5,000～8,000 億ドル規模へ拡大、年平均 10～15% 程度の成長が見込まれている  
（各種調査レポートの中央値ベース）。

**日本国内 EdTech 市場**

日本国内 EdTech 市場は、2021 年の ¥2,674 億円から 2027 年には ¥3,625 億円へ拡大し、  
2021 年比で約 1.3～1.5 倍の成長が見込まれている。  
特にコンテンツ（教科学習）分野が最大シェアを占めている。

### ターゲット市場セグメント

**セグメント 1：中堅 EdTech スタートアップ（優先度 1）**
- 講座数: 10～30 本
- 年間売上: 1～5 億円
- 特徴: データドリブン改善に関心高いが、分析人材が限定的
- 市場規模推定: 50～100 社（日本国内）
- TAM: ¥60～120 M/年（月 100k × 12 月 × 50～100 社）

**セグメント 2：大手オンライン教育企業（優先度 2）**
- 講座数: 30～100+ 本
- 年間売上: 5～100 億円
- 特徴: 既に分析チームあり、高度なカスタマイズニーズ
- 市場規模推定: 20～40 社（日本国内）
- TAM: ¥120～240 M/年（月 100k～200k × 20～40 社）

**セグメント 3：個人クリエイター / 小規模プロダクション（将来対応）**
- 講座数: 1～10 本
- 年間売上: <¥100M
- 特徴: 価格感度高い、まずは無料ツール / 安価プラン希望
- 対応: Phase 2+ で廉価版・フリーミアム計画

### ターゲット TAM の考え方と成長戦略

本提案では、日本国内の「講座数 5～20 本クラス」の EdTech 企業に絞った場合のニッチな TAM を、  
年間ベースで約 **¥60～120M** と見積もる。

中長期的には、企業内研修動画、社内 LMS、金融機関の教育コンテンツ等、  
隣接ドメインへの水平展開により、対象市場の拡大が可能である。  
初期段階での市場規模が限定的であっても、垂直統合による高粗利率運営と、  
段階的な市場拡大により、持続可能な成長を見込む。

### 市場ニーズ根拠

**課題 1: データと経営判断のギャップ**
- 視聴ログは取得できる企業が増加中（90%+）
- しかし「視聴数だけ」では施策判断に不十分（複雑性が高い）
- Excel 集計に週単位の時間消費

**課題 2: 成長テーマの発見遅延**
- 講座間の成長率比較が手作業
- テーマ別分析には時間・人材が必要
- 機会ロスが発生（成長テーマの深掘りまで 1～2 ヶ月遅延）

**課題 3: 組織間の意思決定スピード**
- 営業チーム「どの講座を推すべき？」
- マーケティング「次のコンテンツは？」
- 経営「テーマ別 ROI は？」
- → 統一された数字がないため、結論が出ない

## 3. ビジネスモデル

### サービス提供フロー

**Phase 1: ヒアリング / セットアップ（初月）**
- 顧客の講座構成・テーマ分類を確認（30 分～1h の打ち合わせ）
- JSON スキーマのカスタマイズ（必要に応じて）
- API / CSV 形式でデータ連携を構築
- 初回 Executive Summary + Full Report を生成・納品

**Phase 2: 月次自動配信（以降継続）**
- 毎月末：最新データから Executive Summary + Full Report を自動生成
- メール + PDF / HTML 形式で配信（納期: 翌営業日朝）
- 必要に応じてオンライン説明（15～30 分、任意）

**Phase 3: 継続的な改善提案（付加価値）**
- 月次レビューで顧客の施策実行結果を追跡
- 次月の新テーマ・改善機会を事前提案
- 継続率向上 / アップセル機会の創出

### 競合優位性

**セットアップ時間**

一般的な BI ツールの導入では、要件定義〜ダッシュボード設計を含めると、  
4〜12 週間程度を要するケースが多い一方で、本サービスでは標準的なセットアップを  
1〜2 週間で完了可能な設計としている。

**レポート自動化**

レポート生成プロセスは、標準的なレポート項目について 100% 自動化を実現し、  
人手による再集計・再計算作業をほぼ不要とする設計である。  
（カスタム分析は必要に応じて別途対応）

**比較表**

| 要素 | 本サービス | 競合（Excel / BI ツール） |
|------|-----------|----------------------|
| **セットアップ時間** | 1～2 週間 | 4～12 週間 |
| **レポート自動化** | 原則 100% | 0%（手作業） |
| **判断根拠の一貫性** | 統一（JSON ベース） | バラバラ（人依存） |
| **テーマ別分析** | 自動・リアルタイム | 手作業・遅延 |
| **成長機会の発見** | 早期（週単位） | 遅延（月単位） |
| **総運用コスト（年額）** | ¥1.5 M | ¥5～10 M + BI ツール |

### 運用コスト構造

**固定コスト（月次）**
- クラウドインフラ（AWS / GCP）: ¥20～30 万/月
- チームメンバー（段階拡張による平均）: ¥80～150 万/月
- 合計: ¥100～180 万/月（スケール段階によって変動）

**顧客当たり変動コスト**
- データ処理 / API コール: 顧客 1 社あたり ¥3～5 万/月
- オンライン対応（オプション）: ¥1～2 万/月

**売上ミックス仮定**
- 基本プラン（月 100k）: 70%
- プレミアムプラン（月 200k）: 20%
- エンタープライズ（月 300k+）: 10%

### コスト構造と利益率（シナリオ B ベース）

**Year 1（2026）**

**Year 2（2027）**

**Year 3（2028）**

※ Year 3 は営業・G&A・R&D の重点投資により、利益率を 40～50% 程度の現実的レンジに調整。

### 初期投資と回収シナリオ

初期投資 8,500 万円のうち、およそ 5,000 万円をプロダクト開発・R&D、  
2,000 万円を初期営業・マーケティング、1,500 万円をインフラ・運転資金に充当する想定である。

シナリオ B を前提とした場合、Year 1〜2 でのキャッシュフロー累積により、  
**Year 3〜4 目でキャッシュベースの投資回収を見込む。**

Year 4 以降は、営業利益率 40～50% での自己成長が可能となり、  
追加調達なしでの事業拡大を見込む。

### IP・プロダクト価値

**焦点**: プロダクト完成度向上、パイロット顧客 3～5 社との事例構築、組織基盤整備

Year 2〜3 では営業・カスタマーサクセス体制を拡充し、  
顧客数 30 社・年間売上 4～5 億円規模での黒字化と自己成長を目指す。

この段階で見込まれる EBITDA（¥19～24M）により、Series B 後は外部調達に依存しない成長が可能となる。

**売上**: ¥45M（顧客 30 社）

## 6. リスク & 対策

### リスク 1：顧客獲得の難しさ

**リスク内容**
- EdTech / 講座ビジネスは非常にニッチ市場
- ターゲット顧客（5～20 本クラス）の認知が低い
- 営業効率（CAC: 顧客獲得コスト）が不明確

**対策**
1. **パイロット顧客の事例化**: 初期 3～5 社との成功事例を詳細に記録し、営業資料化
2. **業界パートナーシップ**: EdTech 企業向け支援機関・インキュベーターとの提携による紹介ネットワーク構築
3. **コンテンツマーケティング**: note / ブログで「データドリブン改善の実例」を継続発信
4. **フリートライアル / デモ**: 初期 30 日間の無料トライアル or サンプルレポート無料配信

### リスク 2：顧客データの管理・セキュリティ

**リスク内容**
- 視聴ログは顧客にとって機密情報（講座の人気度など）
- GDPR / 個人情報保護法への準拠が必須
- データ漏洩時の企業信用喪失

**対策**
1. **SOC 2 / ISO 27001 認証**: 初年度内の取得を計画

   SOC 2 / ISO 27001 などの認証取得には、一般的に数百万円規模のコストと  
   6〜12 ヶ月程度の期間を要するが、主要顧客層への信頼性確保のため、  
   優先度の高い投資として位置づける。

2. **暗号化・監査ログ**: エンドツーエンド暗号化 + 全アクセスの監査証跡を実装
3. **定期セキュリティ監査**: 四半期ごとの外部監査実施
4. **利用規約・SLA の明確化**: 99.9% 稼働保証、バックアップ運用の文書化

### リスク 3：プロダクト陳腐化 / 競合追随

**リスク内容**
- 大手 BI ツール（Tableau / Looker）が EdTech 特化版を出す可能性
- 低コスト AI ツールの台頭

**対策**
1. **継続的なプロダクト進化**:
   - Phase 2: AI による自動提案生成
   - Phase 3: リアルタイムダッシュボード
   - Phase 4: REST API + Slack 統合
2. **顧客ロックイン**: 初期導入で 3 ヶ月以上の契約を標準化
3. **差別化資産の強化**: 3 層設計、Representative Pin（代表的学び）の専有化

### リスク 4：初期採用率の鈍化

**リスク内容**
- 「Excel でいい」「BI ツール使ってる」という既存ツール満足層の存在
- 切り替えコスト（導入、学習）の認識

**対策**
1. **ROI シミュレーター**: 顧客が「月あたり何時間削減できるか」を可視化するツール提供
2. **無償コンサル**: 初期 1h のコンサルで顧客のデータ課題を言語化
3. **段階的導入**: 初回は Executive Summary だけ、Full Report は後付けで OK な柔軟性

## 7. 出口戦略（M&A / IPO）

### 想定買い手と シナジー

**シナリオ A：大手 EdTech / オンライン教育企業への買収**
- 例：ベネッセ、レアジョブ、Schoo など
- シナジー: 既存の講座ポートフォリオに対し、本サービスの分析・レポート機能を内製化することで、  
  LTV 向上やアップセル商材としてのシナジーが見込める
- 評価額: ARR の 4～6 倍（Year 3 時点で ¥180～270 M）

  プロダクトラインナップの充実と新規セグメントへの参入が可能となる

**シナリオ C：日本発 SaaS として IPO**
- 上場市場：JQ / 東証グロース（Year 4-5）
- シナジー: データドリブン教育の重要性が認識される社会背景のもと、  
  上場企業としてのブランド構築と国際展開の加速が可能となる
- 評価額: ARR の 8～10 倍（¥360～450 M）

## 付録：定義・用語集

- **Executive Summary**: 社長が 3 分で読めるサイズの月次レポート（1 枚）
- **Full Report**: 施策立案用の詳細レポート（Portfolio / Growth / Theme View）
- **3 レイヤー JSON**: video_meta（講座定義）/ knowledge_core（学習内容）/ views（視聴データ）
- **ARPU**: 顧客あたり月額売上（Average Revenue Per User）
- **ARR**: 年間経常収益（Annual Recurring Revenue）
- **CAC**: 顧客獲得コスト（Customer Acquisition Cost）
- **TAM**: 総市場規模（Total Addressable Market）
- **EBITDA**: 利息・税金・減価償却前利益（Earnings Before Interest, Taxes, Depreciation and Amortization）
- **PMF**: プロダクトマーケットフィット（Product Market Fit）
