# ユーザーガイド：PoC LP・サンプルレポートの見方と使い方

## 概要
このガイドは、video-insight-spec の PoC LP とサンプルレポート（Markdown/HTML版）を初めて使用するユーザーを対象としています。

---

## セクション 1：PoC LP について

### 1-1. PoC LP とは？
PoC LP（Proof of Concept Landing Page）は、EdTech スタートアップ向けのオンライン講座分析レポート機能を紹介するランディングページです。

**主な特徴：**
- 講座の視聴数・エンゲージメント率をダッシュボード表示
- テーマ別・講座別の成長トレンド可視化
- 営業チーム向けのアクション提案を自動生成

### 1-2. PoC LP へのアクセス方法
1. メール内のリンク、または営業担当者から提供される URL にアクセス
2. ブラウザで PoC LP のランディングページが表示されます
3. 「詳細レポートを見る」ボタンで、サンプルレポートへ遷移

### 1-3. PoC LP で確認できる内容
- **総視聴数・前月比**：講座全体の視聴トレンド
- **平均エンゲージメント率**：視聴者の関心度合い
- **成長テーマ Top 3**：最も伸びている講座テーマ
- **推奨アクション**：次月に取るべき施策

## セクション 3：レポート数値の定義

### 3-1. 総視聴数
**定義：** 対象期間（過去 1 ヶ月）に、各講座が被観視した回数の合計

### 3-2. 前月比
**定義：** 今月の総視聴数を先月の総視聴数で割り、パーセンテージで表示
例：今月 280,000 回 / 先月 250,000 回 = +12% 相当

### 3-3. 平均エンゲージメント率
**定義：** 視聴完了率、リプレイ率、課題提出率などを総合的に算出した、視聴者の関心度スコア

目安：
- 1.5% 以上：高い関心度（優良コンテンツ）
- 1.0～1.5%：中程度の関心度
- 1.0% 未満：改善余地あり

### 3-4. テーマ別成長率
**定義：** 特定テーマの講座群（例：「Python」「SQL」など）の視聴数前月比

## セクション 4：よくある質問（FAQ）

### Q1. サンプルレポートのデータは実際の顧客データですか？
**A：** いいえ。サンプルレポートは、**仮想データ**を基に構成しています。実装時（Phase 7 以降）は、顧客の実績データを匿名化した上で、このテンプレートに置き換える予定です。

### Q2. レポートはどのくらいの頻度で更新されますか？
**A：** 毎月末時点での直前 1 ヶ月間のデータを集計し、翌月初に新規レポートを生成することを想定しています。

### Q3. エンゲージメント率は独自の定義ですか？
**A：** はい。video-insight-spec では、完了率・リプレイ率・課題提出率などを加重平均して算出した独自スコアです。詳細な計算式についてはサポートまでお問い合わせください。

### Q4. 「推奨アクション」の根拠は？
**A：** 成長テーマ、完了率、転換率などのメトリクスを組み合わせ、営業チームの経験則と統計的根拠に基づいて提案しています。

### Q5. HTML 版と Markdown 版の違いは？
**A：** 機能的には同一です。HTML 版はブラウザでそのまま表示可能、Markdown 版はテキストエディタで編集・カスタマイズが容易です。用途に応じて選択ください。
