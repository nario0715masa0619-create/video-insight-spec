# 営業ガイド：サンプルレポートを活用した提案フロー

## 概要
このガイドは、営業チームがサンプルレポートを使用して、リード獲得から提案・成約までを進める際の実践的な手順をまとめたものです。

## セクション 2：各ステップの詳細

### ステップ 1：リード獲得
**主な獲得チャネル：**
- インバウンド（Web サイト、SEO）
- 広告（Google Ads、LinkedIn）
- パートナー紹介

**初期接触時のポイント：**
- 顧客の現状課題（講座視聴数の伸び悩み、エンゲージメント低下など）をヒアリング
- 顧客の業種・講座数・月額売上規模を把握

---

### ステップ 2：PoC LP で関心喚起

**目的：** 顧客に「このツールで何ができるか」を視覚的に理解してもらう

**提供方法：**
1. メール内に PoC LP 用 URL を記載
2. または面談時にプロジェクタで PoC LP のデモを見せる

**PoC LP で強調すべきポイント：**
- リアルタイムダッシュボード：視聴数・エンゲージメント率の可視化
- 自動インサイト生成：データから営業アクション提案を自動抽出
- 3 種類のレポートテンプレート：業種別・講座タイプ別の分析フレーム

**サンプル話法：**
> 「video-insight-spec は、EdTech スタートアップの講座分析を自動化するツールです。毎月のレポート作成時間を 80% 削減し、営業チームへの提案を 2 週間早く提供できます。まずはサンプルレポートをご覧ください。」

---

### ステップ 3：顧客業種に合わせてサンプルレポートを選択

**業種別サンプルレポート選択表：**

| 顧客業種 | 推奨サンプルレポート | 理由 |
|---------|-------------------|------|
| SNS・マーケティング講座提供社 | マーケティング教材系 | SNS テーマの成長戦略が参考になる |
| Web 制作・プログラミング講座提供社 | Web 制作系 | JavaScript・デザイン講座の実例が活用できる |
| データ分析・統計講座提供社 | データ分析系 | Python・SQL の転換率分析が参考になる |
| 複数業種（統合学習プラットフォーム） | 全 3 種類を順序よく説明 | 幅広い分析フレームの提供が効果的 |

#### 面談提供パターン

**提示タイミング：** 2 回目以降の商談（1 回目でスクリーニング完了後）

**提示方法：**
1. プロジェクタまたは共有画面で HTML 版レポートを表示
2. Executive Summary から始めて、上から順に説明
3. 顧客の業種・講座数に応じて、該当セクションを詳しく解説

**説明の進め方：**
- 「このレポートは、毎月自動で生成されます」と伝え、作業削減効果を示唆
- グラフやテーブルを指しながら、「貴社の場合は…」と仮定の話をする
- 「実際には顧客データで同じ分析をします」と、期待値を高める

### ステップ 5：質問・ご要望を引き出す

**想定される顧客質問と対応：**

| 顧客質問 | 営業対応 |
|---------|--------|
| 「レポート作成にはどのくらい時間がかかりますか？」 | 「テンプレート適用で 2～3 時間。従来の 12 時間から 80% 削減です。」 |
| 「データセキュリティは大丈夫ですか？」 | 「GDPR・個人情報保護対応済み。詳しくはセキュリティホワイトペーパーをご参照ください。」 |
| 「導入後、本当に売上が上がりますか？」 | 「既存顧客では月 15～20% の視聴数増を実現。推奨アクションの実装が成功の鍵です。」 |
| 「他社の分析ツールとの違いは？」 | 「業界専用テンプレート、自動提案生成、営業向けアクション重視が特徴です。」 |

**質問が出ない場合の切り口：**
- 「このレポートで一番興味深いと感じた部分はどこですか？」
- 「実際に貴社のデータで分析した場合、どの指標に注目したいですか？」
- 「現在、どのようなツールでレポート作成をされていますか？」

---

### ステップ 6：実績データを用いた提案資料を作成

**プロセス：**
1. 顧客より過去 3 ヶ月分の講座視聴データを収集
2. video-insight-spec のテンプレートに顧客データを適用
3. カスタマイズされた提案レポートを作成
4. 営業メッセージ（推奨アクション、期待効果）を追加

**期待効果の説き方：**
\\\
貴社の場合：
- 現在の月平均視聴数：50,000 回
- 推奨アクション実装による予想増：10～15%
- 期待値：55,000～57,500 回 / 月

売上への波及効果：
- 1 講座あたりの平均価格：¥10,000
- 視聴数増 → 購買転換率向上（従来 2% → 推定 2.3%）
- 月間売上インパクト：¥250,000～¥375,000
\\\

### ステップ 7：最終提案・契約

**最終プレゼンの構成：**
1. 現状分析レポート（顧客の実績データ活用）
2. 推奨施策と期待効果（定量化された成果）
3. 導入プラン・タイムライン（フェーズ 1、2）
4. 価格・ご契約条件
5. Q&A

**契約前の確認チェックリスト：**
- [ ] 顧客の意思決定者が全員説明を聞いたか
- [ ] 予算承認プロセスを確認したか
- [ ] 導入開始時期の目安を取り決めたか
- [ ] サポート体制（オンボーディング、定期レビュー）を説明したか

## セクション 3：成約事例・テンプレート

### 事例 1：SNS マーケティング教材社

**顧客背景：**
- 講座数：6 本
- 月間売上：約 ¥3,000,000
- 課題：SNS トレンドへの対応が遅れ、視聴数が伸び悩んでいた

**提案内容：**
- SNS マーケティング教材系サンプルレポートを基に、カスタマイズ提案
- 週 1 本の新規 SNS テーマ企画、既存講座との統合パッケージ化

**成果：**
- 導入後 3 ヶ月で視聴数 +18%、売上 +22%
- レポート作成時間を月 10 時間削減

---

### 事例 2：Web 制作系スクール

**顧客背景：**
- 講座数：12 本
- 月間売上：約 ¥8,000,000
- 課題：講座多数で、どの講座に投資すべきか判断困難

**提案内容：**
- Web 制作系サンプルレポートで、講座別の成長性を可視化
- JavaScript 中級・デザイン講座の拡充を推奨

**成果：**
- 導入後 2 ヶ月で認知率 +12%、完了率 +8%
- マーケティング ROI が 35% 向上

## セクション 4：サンプルレポート活用チェックリスト

提案時に以下の項目を確認し、顧客の納得度を高めましょう：

- [ ] Executive Summary が 1 ページで理解できるか説明したか
- [ ] 成長テーマ Top 3 の根拠（数値）を顧客が理解したか
- [ ] Top 3 講座の詳細を、顧客の講座群と比較説明したか
- [ ] 推奨アクションが「実装可能で、効果が見込める」と納得させたか
- [ ] 期待効果の数値が現実的か、顧客から質問が出たか
- [ ] インサイトが顧客の課題解決に直結していることを示したか
- [ ] HTML・Markdown 両方の活用シーンを説明したか

## セクション 5：よくある営業質問と対応

### Q1. 「サンプルレポートのデータはどこから取得していますか？」
**A：** 業界平均・既存顧客データ・パブリックデータを参考に、仮想構成しています。実装後は顧客のアクチュアルデータを使用します。

### Q2. 「導入にはどのくらいの準備期間が必要ですか？」
**A：** データ接続の設定で 1～2 週間。運用開始から最初のレポート提出まで 1 ヶ月が目安です。

### Q3. 「レポートはカスタマイズできますか？」
**A：** はい。セクション追加、メトリクス変更、デザイン調整が可能です。大規模カスタマイズはオプション費用が発生します。

### Q4. 「複数の顧客に同じサンプルレポートを見せても大丈夫ですか？」
**A：** はい。業種別サンプルがあるため、該当業種のレポートであれば問題ありません。ただし「仮想データのサンプル」である旨は明示してください。
