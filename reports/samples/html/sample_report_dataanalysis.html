代表的インサイト

インサイト 1：初心者向けコンテンツの需要が高い

統計学基礎（ノーコード）講座の視聴数は少ないが、完了率 72% で継続学習への関心が高い。→ 初級者向け充実化で新規ユーザー流入期待

インサイト 2：Python＋SQL セット化による相乗効果

Python 講座視聴後の SQL 講座転換率が 32%（業界平均 ~25% 想定）で高い。→ セット販売・カリキュラム統合で相乗効果見込める可能性

インサイト 3：可視化ツール供給不足

Tableau 講座の視聴は増加傾向だが完了率は 48% で低迷。→ 初級～中級別、ツール別（Power BI・Looker も含む）講座拡充で需要対応可能

インサイト 4：R 言語講座は需要縮小傾向

R 言語講座は視聴数が低下（-0.8% MoM）。Python への置き換わり進行中か。→ R 講座廃止検討、または R/Python 比較講座に統合
